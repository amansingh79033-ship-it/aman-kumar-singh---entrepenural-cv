# Audio Assets

This directory contains audio assets for the application.

## Required Assets

- `sufi-flute-sitar.mp3` - Background music with Sufi flute and sitar for poetry playback

## Instructions

To implement the full functionality, add a high-quality audio file named `sufi-flute-sitar.mp3` to this directory. The audio should feature:

- Soulful Sufi flute melodies
- Complementary sitar accompaniment
- Loopable format for continuous playback
- Mild volume level (around 20%)
- Duration of at least 30 seconds for seamless looping
